## 🔐 基础概念

### Q1: 什么是非托管钱包？

**A**: 非托管钱包是指用户100%控制私钥的钱包。

**关键特点**:
- ✅ 私钥只在你的设备上
- ✅ 平台无法访问你的资产
- ✅ 平台被黑，你的资产仍然安全
- ⚠️ 你必须自己保管助记词

**对比**:
```
托管钱包（如交易所）:
  平台控制私钥 → 平台控制资产 → 需要信任平台

非托管钱包（如IronForge）:
  你控制私钥 → 你控制资产 → 无需信任平台
```

---

### Q2: 助记词是什么？

**A**: 助记词是12或24个单词，用于恢复钱包的唯一方式。

**示例**（12个单词）:
```
abandon ability able about above absent 
absorb abstract absurd abuse access accident
```

**重要性**:
- 🔑 助记词 = 你的所有资产
- ⚠️ 任何人获得助记词 = 可以盗取所有资产
- ⚠️ 丢失助记词 = 资产永久丢失

---

### Q3: 为什么有两个密码？

**A**: 系统使用双锁机制，两个密码用途不同。

| 密码 | 用途 | 可否重置 | 安全级别 |
|------|------|----------|----------|
| **账户密码** | 登录平台 | ✅ 可重置（邮箱验证） | 中等 |
| **钱包密码** | 解锁钱包、签名交易 | ❌ 不可重置（需助记词恢复） | 极高 |

**使用场景**:
```
登录平台 → 需要账户密码
查看余额 → 不需要钱包密码
发送交易 → 需要钱包密码 ⚠️
```

## 🔒 密码管理

### Q4: 我忘记账户密码怎么办？

**A**: 可以通过邮箱重置。

**步骤**:
1. 点击"忘记密码"
2. 输入注册邮箱
3. 收取验证邮件
4. 点击邮件链接
5. 设置新密码
6. ✅ 重置成功

**影响**:
- ✅ 可以重新登录平台
- ✅ 可以查看钱包列表和余额
- ⚠️ 如果忘记钱包密码，仍无法发送交易

---

### Q5: 我忘记钱包密码怎么办？

**A**: 需要使用助记词恢复钱包。

**场景1：已备份助记词** ✅
```
1. 使用"恢复钱包"功能
2. 输入备份的助记词（12或24个单词）
3. 设置新的钱包密码
4. ✅ 钱包恢复，可以使用新密码
```

**场景2：没有备份助记词** ❌
```
⚠️ 资产永久丢失
❌ 无法找回
💡 这就是非托管钱包的代价
```

**预防措施**:
- ✍️ 创建钱包时立即备份助记词
- 📝 手写到纸上（不要打印）
- 🔒 存放到安全地方（保险柜）

---

### Q6: 我丢失了助记词怎么办？

**A**: ⚠️ 无法找回，请立即转移资产。

**紧急措施**:
1. ⚡ 如果钱包仍然解锁（或记得钱包密码）
2. ⚡ 立即创建新钱包
3. ⚡ 妥善备份新钱包的助记词
4. ⚡ 将旧钱包的所有资产转移到新钱包
5. ⚡ 废弃旧钱包

**后果**:
- ❌ 丢失钱包密码+助记词 = 资产永久丢失
- ❌ 平台无法帮你找回
- ❌ 这是非托管钱包的本质

## 🎯 钱包操作

### Q7: 如何创建钱包？

**A**: 3个步骤。

**步骤**:
```
1. 注册账户
   输入：邮箱 + 账户密码
   
2. 创建钱包
   输入：钱包名称 + 钱包密码
   系统生成：24个单词的助记词
   
3. 备份助记词 ⚠️ 重要！
   ✅ 手写到纸上
   ✅ 抄写2-3份
   ✅ 存放到不同安全地点
   ✅ 验证3个随机单词
```

**时间**: 约5分钟

---

### Q8: 如何发送交易？

**A**: 需要解锁钱包。

**完整流程**:
```
1. 登录账户（账户密码）
2. 进入"发送"页面
3. 输入接收地址和金额
4. 点击"发送"
5. 弹窗要求输入钱包密码 ⚠️
6. 输入钱包密码解锁
7. 本地签名交易 🔐
8. 发送到区块链
9. ✅ 交易完成
```

**安全提示**:
- 🔐 交易在您的设备上签名
- 🔐 私钥不会上传到服务器
- 🔐 钱包密码不会发送给平台

---

### Q9: 钱包会自动锁定吗？

**A**: 是的，15分钟无操作自动锁定。

**锁定机制**:
- ⏱️ 15分钟无操作 → 自动锁定
- 🔄 每次签名交易 → 重置计时器
- 🔒 锁定后 → 再次签名需要钱包密码

**好处**:
- 防止他人使用你的设备发起交易
- 即使设备被盗，钱包仍然安全

| 钱包密码 | ❌ 不存储 | ❌ 绝不存储 |
| 地址 | ✅ 存储 | ✅ 存储（公开信息） |

**平台可以做的**:
- ✅ 查询你的余额（使用公开地址）
- ✅ 广播你的交易（你已签名）
- ❌ 不能访问你的私钥
- ❌ 不能动用你的资产

## 💰 交易相关

### Q15: 为什么交易需要Gas费？

**A**: Gas费是支付给区块链矿工的手续费。

**Gas费组成**:
```
总Gas费 = Gas价格 × Gas限制

示例（Ethereum）:
  Gas价格: 50 Gwei
  Gas限制: 21,000（普通转账）
  总Gas费: 50 × 21,000 = 1,050,000 Gwei = 0.00105 ETH
```

**谁收取**:
- ✅ 区块链矿工（或验证者）
- ❌ 不是IronForge平台

**优化建议**:
- 非紧急交易：使用低Gas价格（节省费用）
- 紧急交易：使用高Gas价格（快速确认）

---

### Q16: 交易pending很久，怎么办？

**A**: 可以使用交易加速功能。

**步骤**:
```
1. 进入"交易历史"
2. 找到pending的交易
3. 点击"加速交易"
4. 系统建议新的Gas价格（+10%以上）
5. 输入钱包密码重新签名
6. ✅ 新交易会替换旧交易
```

**原理**: Replace-by-Fee (RBF)
- 使用相同的nonce
- 提高Gas价格
- 区块链优先打包高Gas交易

---

### Q17: 如何取消pending的交易？

**A**: 发送0金额交易到自己地址。

**步骤**:
```
1. 记录pending交易的nonce（如：nonce=5）
2. 创建新交易：
   - 接收地址：你自己的地址
   - 金额：0
   - nonce：5（与pending交易相同）
   - Gas价格：比原交易高10%
3. 签名并发送
4. ✅ 新交易替换原交易
```

## 🌉 跨链相关

### Q18: 跨链转账需要多长时间？

**A**: 通常10-30分钟。

**时间分解**:
```
源链确认：5-10分钟
跨链桥处理：3-10分钟
目标链确认：2-10分钟
━━━━━━━━━━━━━━━━
总计：10-30分钟
```

**影响因素**:
- 源链网络拥堵
- 目标链网络拥堵
- 跨链桥性能

---

### Q19: 跨链失败了，钱会丢失吗？

**A**: ❌ 不会，资产仍在源链。

**失败场景**:
1. 源链交易确认，但目标链失败
   - 资产在跨链桥合约
   - 可以申请退款
   
2. 源链交易失败
   - 资产仍在你的钱包
   - 只损失Gas费

**安全机制**:
- 源链使用锁定机制（不是销毁）
- 24小时内未完成可退款
- 跨链桥有资产保险

## 🎓 高级话题

### Q24: BIP39/BIP44是什么？

**A**: 行业标准的助记词和派生协议。

**BIP39**:
- 将随机数转换为助记词
- 12个单词 = 128 bit熵
- 24个单词 = 256 bit熵

**BIP44**:
- 从一个助记词派生多条链的钱包
- 标准路径格式
- 跨钱包兼容

**示例路径**:
```
Ethereum:  m/44'/60'/0'/0/0
Bitcoin:   m/84'/0'/0'/0/0
Solana:    m/44'/501'/0'/0'
```

---

### Q25: 为什么不同钱包派生的地址相同？

**A**: 因为使用相同的标准协议。

**原理**:
```
相同输入（助记词） + 相同算法（BIP44） + 相同路径
→ 相同输出（地址）
```

**例子**:
```
你在IronForge创建钱包：
  助记词: abandon ability able...
  以太坊地址: 0x742d35Cc...

你在MetaMask导入相同助记词：
  助记词: abandon ability able...
  以太坊地址: 0x742d35Cc...  ✅ 相同！
```
