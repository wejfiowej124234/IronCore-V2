## ✅ 非托管原则（必须遵守）

- 后端不接收助记词/私钥/钱包密码
- 客户端本地生成/导入助记词并加密存储，本地派生地址/公钥与交易签名
- 后端仅接收公开信息（例如地址、公钥）与业务记录
