## 📖 技术栈

### 核心框架
- **Axum** - Web 框架
- **Tokio** - 异步运行时
- **SQLx** - 数据库驱动
- **Redis** - 缓存层
- **Immudb** - 审计日志

### 数据库
- **CockroachDB** / **PostgreSQL** - 主数据库
- 端口: 26257 (CockroachDB) / 5432 (PostgreSQL)
- 连接池: 最大 20 连接

### API 规范
- RESTful API
- OpenAPI 3.0 规范
- 文档地址: `http://localhost:8088/openapi.yaml`
