## 🔑 助记词管理

### ✅ 必须做的事

#### 1. 手写备份（推荐方式）

**步骤**:
```
1. 准备：
   - 优质纸张（不易损坏）
   - 防水笔
   - 安静的环境（无监控摄像头）

2. 抄写：
   - 按顺序抄写24个单词
   - 标注序号（1-24）
   - 使用清晰字迹

3. 验证：
   - 逐个核对拼写
   - 确认顺序正确
   - 尝试输入到钱包验证

4. 存放：
   - 密封袋（防水）
   - 保险柜或银行保管箱
   - 告知信任的家人位置（不告诉内容）
```

---

#### 2. 金属备份（最安全方式）

**推荐产品**:
- Cryptosteel Capsule
- Billfodl
- 不锈钢板 + 刻字

**优势**:
- ✅ 防火（可承受1400°C）
- ✅ 防水
- ✅ 防腐蚀
- ✅ 长期保存（50年以上）

**成本**: $50-$200

---

#### 3. 多地备份（分散风险）

**建议方案**:
```
备份1: 家里保险柜
备份2: 银行保管箱
备份3: 信任的家人处（密封信封）
```

**注意**:
- ❌ 不要把所有备份放一个地方
- ❌ 不要把所有助记词写在同一张纸

**分割存储（高级）**:
```
使用Shamir's Secret Sharing（助记词分割）:
  - 将助记词分割为3份
  - 任意2份可以恢复
  - 单份无法恢复

工具: ssss-cli, SLIP-39
```

---

### ❌ 绝对不要做的事

#### 1. ❌ 数字化存储

**危险方式**:
- ❌ 截图保存
- ❌ 手机拍照
- ❌ 电脑文档
- ❌ 云端存储（iCloud/Google Drive/Dropbox）
- ❌ 密码管理器（如1Password）
- ❌ 笔记应用（Evernote/Notion）

**原因**:
- 云同步可能泄露
- 设备被黑可能泄露
- 云服务商可能被黑

**例外**: 如果必须数字化存储
```
1. 使用加密U盘（硬件加密）
2. 离线存储（永不联网）
3. 放入保险柜
```

---

#### 2. ❌ 网络传输

**危险方式**:
- ❌ 微信/QQ发送
- ❌ 邮件发送
- ❌ Telegram/WhatsApp
- ❌ 任何聊天工具
- ❌ 短信

**原因**:
- 网络传输可能被拦截
- 服务器可能保留记录
- 接收方设备可能被黑

---

#### 3. ❌ 告诉他人

**危险场景**:
- ❌ 告诉朋友（朋友可能泄露）
- ❌ 告诉客服（真客服不会要求）
- ❌ 告诉技术支持
- ❌ 在公共场合讨论

**唯一例外**: 
- ✅ 遗产规划：告诉继承人如何找到备份（不告诉助记词本身）

## 🔐 密码安全

### 强密码要求

**账户密码（平台登录）**:
```
✅ 好例子:
  MyIronForge2025@Secure
  
  - 12位以上
  - 大小写字母
  - 数字
  - 特殊符号

❌ 坏例子:
  password123
  
  - 太短
  - 太简单
  - 常见密码
```

**钱包密码（签名交易）**:
```
✅ 好例子:
  Wallet!2025#Secure$Forever
  
  - 16位以上
  - 多种字符类型
  - 独特且难猜

❌ 坏例子:
  12345678
  
  - 太弱
```

---

### 密码管理建议

1. **不要重复使用**
   ```
   ❌ 账户密码 = 钱包密码
   ❌ 钱包密码 = 邮箱密码
   
   ✅ 每个系统使用不同密码
   ```

2. **密码存储**
   ```
   ✅ 使用密码管理器（如1Password, Bitwarden）
   ✅ 手写到纸上（和助记词分开存放）
   
   ❌ 浏览器保存密码（不安全）
   ```

3. **定期更换**
   ```
   账户密码：每3-6个月更换
   钱包密码：每6-12个月更换
   ```

## 💻 设备安全

### 基础防护

1. **操作系统**
   ```
   ✅ 保持最新版本
   ✅ 启用自动更新
   ✅ 使用官方系统（非破解版）
   ```

2. **防病毒软件**
   ```
   Windows: Windows Defender + Malwarebytes
   macOS: 内置安全功能 + ClamAV
   Linux: ClamAV + rkhunter
   ```

3. **浏览器安全**
   ```
   ✅ 使用Chrome/Firefox/Edge最新版
   ✅ 启用HTTPS-Only模式
   ✅ 安装uBlock Origin（广告拦截）
   ✅ 不安装未知插件
   ```

---

### 高级防护

1. **专用设备**
   ```
   理想方案：
     购买一台专门用于加密货币的设备
     - 不上社交网站
     - 不下载其他软件
     - 只用于钱包操作
   
   成本：$300-$500（旧笔记本即可）
   ```

2. **硬件钱包**
   ```
   推荐产品：

     IronForge + 硬件钱包
   ```

3. **冷钱包方案**

   ```
     - 离线设备生成钱包
   ```

## 🎯 交易安全

### 发送交易前检查

**必须验证**:
```
✅ 接收地址
  - 复制粘贴（不要手打）
  - 核对前6位和后4位
  - 发送小额测试

✅ 金额
  - 仔细核对小数点
  - 1.5 ETH ≠ 15 ETH

✅ Gas费
  - 检查是否合理
  - Ethereum: $1-$20（正常）
  - BSC: $0.1-$1（正常）

✅ 合约交互
  - 确认合约地址正确
  - 确认授权金额合理
  - 不授权无限额度
```

---

### 防止剪贴板攻击

**攻击方式**:
```
你复制：0x742d35Cc6634C0532925a3b844Bc9e7595f0bFd2
病毒替换：0xAttackerAddress（攻击者地址）
你粘贴：0xAttackerAddress
→ 资产转给攻击者 ❌
```

**防御措施**:
```
1. 使用防病毒软件
2. 粘贴后再次核对地址
3. 核对前6位和后4位
4. 先发送小额测试
```

## 🚨 攻击防御

### 常见攻击类型

#### 1. 钓鱼网站

**攻击方式**:
- 伪装官方网站
- 诱导输入助记词
- 立即盗取资产

**防御**:
- ✅ 只从书签进入
- ✅ 检查网址
- ✅ 检查SSL证书

---

#### 2. 恶意浏览器插件

**攻击方式**:
- 读取LocalStorage
- 监听clipboard
- 修改网页内容

**防御**:
- ❌ 不安装未知插件
- ✅ 定期审查已安装插件
- ✅ 只从官方商店安装

---

#### 3. 社会工程

**攻击方式**:
- 假冒客服
- 声称"系统升级需要验证"
- 要求提供助记词

**防御**:
- ❌ 客服永远不会要求助记词
- ✅ 主动联系官方客服确认
- ✅ 举报诈骗行为

---

#### 4. 硬件Keylogger

**攻击方式**:
- 物理设备记录键盘输入
- 在网吧或公共电脑上

**防御**:
- ❌ 不在公共电脑使用钱包
- ✅ 使用虚拟键盘（屏幕键盘）
- ✅ 检查USB端口是否有异常设备

## 🎓 进阶主题

### 助记词编码（高级用户）

```
方法：
  将助记词转换为数字编码
  
  abandon → 0001
  ability → 0002
  ...
  
  24个单词 → 24组4位数字
  
  存储：
    可以伪装成电话簿、密码本等
    
风险：
  编码规则泄露 → 助记词泄露
  建议使用标准BIP39词表
```

---

### 社会恢复（Social Recovery）

```
原理：
  将助记词分割为N份
  需要M份可以恢复（M < N）
  
示例（3/5方案）：
  - 分给5个信任的人
  - 任意3人可以恢复
  - 单个人无法恢复
  
工具：
  - Shamir's Secret Sharing
  - SLIP-39标准
```
