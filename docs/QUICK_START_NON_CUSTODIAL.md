## 🎯 5分钟理解非托管钱包

### 什么是非托管钱包？

```
传统银行：
  你 → 存款 → 银行控制 → 银行可以冻结/限制

非托管钱包：
  你 → 助记词 → 100%控制 → 无人可以冻结
```

**核心概念**:
- 🔑 **助记词 = 你的资产**
- 🔐 只有你知道助记词
- 🚫 平台无法访问你的资产
- ⚠️ 丢失助记词 = 永久丢失资产

---

## 🚀 3步开始使用

### Step 1: 注册账户（1分钟）

```
1. 打开 https://ironforge.io
2. 点击"注册"
3. 输入：
   - 邮箱: your@email.com
   - 账户密码: MyAccount123!  （用于登录平台）
4. 验证邮箱
5. ✅ 注册完成
```

**注意**: 账户密码只用于登录，和链上资产无关

---

### Step 2: 创建钱包（2分钟）

```
1. 登录后点击"创建钱包"
2. 输入：
   - 钱包名称: My Wallet
   - 钱包密码: MyWallet@2025!  （用于签名交易）
3. 系统生成24个单词的助记词
4. ⚠️ 仔细抄写到纸上：
   
   abandon ability able about above absent
   absorb abstract absurd abuse access accident
   ... (24个单词)
   
5. 验证：输入3个随机单词
6. ✅ 钱包创建完成
```

**⚠️ 重要**: 助记词只显示一次，必须立即备份！

---

### Step 3: 备份助记词（2分钟）

```
1. 准备纸和笔 ✍️
2. 抄写24个单词（按顺序）
3. 检查拼写和顺序
4. 制作2-3份备份
5. 存放到安全地点：
   - 家里保险柜
   - 银行保管箱
   - 信任的家人处
6. ❌ 不要截图
7. ❌ 不要拍照
8. ❌ 不要存到云端
```

### 发送交易

```
1. 点击"发送"
2. 输入：
   - 接收地址: 0x742d35Cc...
   - 金额: 10 USDT
   - Gas价格: 自动估算（可调整）
3. 点击"发送交易"
4. ⚠️ 弹窗要求输入钱包密码
5. 输入钱包密码
6. 🔐 本地签名（私钥不上传）
7. 发送到区块链
8. ✅ 交易完成（2-5分钟确认）
```

**安全提示**:
- 🔐 签名在你的设备完成
- 🔐 私钥不会离开你的设备
- 🔐 平台只负责广播已签名交易

## 🔒 安全要点

### ✅ 必须做的事

1. **立即备份助记词**
   - 手写到纸上（不要打印）
   - 制作多份备份
   - 存放到不同地点

2. **使用强密码**
   - 账户密码：8位以上
   - 钱包密码：12位以上，包含大小写、数字、符号

3. **定期检查**
   - 确认助记词备份仍然安全
   - 检查设备无病毒

4. **保持警惕**
   - 仔细检查网址（防钓鱼）
   - 不点击可疑链接

---

### ❌ 绝对不要做的事

1. ❌ **截图助记词**
   - 截图可能同步到云端（iCloud/Google Photos）
   - 云账户被黑 → 助记词泄露

2. ❌ **通过网络发送助记词**
   - 微信/QQ/邮件都不安全
   - 可能被拦截

3. ❌ **告诉任何人助记词**
   - 包括客服（真客服不会要求）
   - 包括朋友（可能无意泄露）

4. ❌ **在非官方网站输入助记词**
   - 钓鱼网站会立即盗取
   - 仔细核对网址

## 🆘 紧急情况

### 设备丢失

```
1. 立即在新设备恢复钱包
2. 使用助记词导入
3. ⚡ 立即转移所有资产到新钱包
4. 原因：担心设备被破解
```

---

### 怀疑助记词泄露

```
1. ⚡ 立即创建新钱包
2. ⚡ 转移所有资产
3. 废弃旧钱包
4. 时间就是生命！
```

---

### 平台有问题

```
1. 打开MetaMask或Trust Wallet

3. ✅ 资产完全恢复

5. 这就是非托管的优势！
```
